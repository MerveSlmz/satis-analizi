## Analiz Özeti

- 541.909 satır ve 8 değişkenden oluşan e-ticaret verisi analiz edilmiştir.
- Günlük satış trendleri zaman serisi analizi ile incelenmiştir.
- Ülke bazlı analizde Birleşik Krallık toplam cironun büyük kısmını oluşturmuştur.
- Ürün bazlı analizde az sayıda ürünün toplam satışın büyük bölümünü getirdiği görülmüştür.
- Kargo (postage) gelirlerinin toplam ciroda önemli bir payı olduğu tespit edilmiştir.
